## About

UNXHub started as a personal project to explore LUAU and learn new things.  
It is fully open-source. You can use it, modify it, or take parts of the code for your own projects. No credit is needed.  

It is meant to be a simple way to experiment and learn.


| Feature | Tab/Category | Description |
|:---|:---|:---|
| **Walkspeed** | Main / Character | Adjusts the player's movement speed (range shown: 16/500). |
| **JumpPower** | Main / Character | Modifies the player's jump height (range shown: 50/1000). |
| **Max Zoom** | Main / Character | Sets the maximum camera zoom distance (range shown: 400/1000). |
| **Gravity** | Main / Character | Controls the player's gravitational force (value shown: 196.199...). |
| **Infinite Jump** | Main / Character | Allows continuous jumping without needing to touch the ground. |
| **Noclip** | Main / Character | Allows the player to pass through solid objects (walls, terrain). |
| **Force Third Person** | Main / Character | Locks the player's camera into a third-person view. |
| **X-Ray / X-Ray Transparency (%)** | Main / Character | Renders parts of the environment transparent for visibility (transparency shown: 80%/100%). |
| **Fly / Fly Speed** | Main / Fly | Enables free movement through the air and controls the speed of flight. |
| **Reset Character** | Main / Misc | Resets the character's appearance and status. |
| **Reset Walk Speed/Jump Power/Max Zoom/Gravity** | Main / Misc | Resets the individual Character settings to their default values. |
| **Field Of View (FOV)** | Visuals / Game | Adjusts the player's visible angle of the game world (range shown: 70/120). |
| **Full Bright / No Fog** | Visuals / Game | Modifies lighting and atmospheric effects for better visibility. |
| **ESP** | Visuals / ESP | Enables general Extra Sensory Perception (seeing entities through walls). |
| **Outline** | Visuals / ESP | Draws an outline around players/entities. |
| **Tracers** | Visuals / ESP | Draws lines from the player to targets. |
| **Outline Fill/Outline Transparency (%)** | Visuals / UI | Controls the transparency of the outline/fill colors. |
| **Show Distance / Show Player Name** | Visuals / UI | Toggles displaying the distance and name of other players. |
| **Tracers Position** | Visuals / UI | Sets the anchor point for tracers (e.g., Down). |
| **Rainbow ESP/Outline/Tracers** | Visuals / ESP Config | Toggles color cycling effects for the visual features. |
| **Rainbow Speed** | Visuals / ESP Config | Controls the speed of the color cycling effect. |
| **ESP Size / ESP Font** | Visuals / ESP Config | Configures the size and font style of ESP text. |
| **Enable Aimlock** | Features / Aimlock | Toggles the automatic aiming feature. |
| **Aimlock Type** | Features / Aimlock | Selects the target priority (e.g., Nearest Character). |
| **Wall Check / Team Check** | Features / Aimlock | Options to refine targeting (ignore targets behind walls or on the same team). |
| **Aimlock Certain Player** | Features / Aimlock | Allows specifying a single target player for aimlock. |
| **Aimlock Max Dist / Mouse Max Dist / FOV Max Dist** | Features / Aimlock Config | Sets the maximum range for targeting based on various metrics. |
| **Smooth Aimlock / Aimbot Smoothness** | Features / Aimlock Config | Configures how smoothly the aim correction occurs. |
| **Aimlock Offset (X/Y)** | Features / Aimlock Config | Allows fine-tuning the target location (e.g., head or chest). |
| **Whitelist Players / Prioritize Players** | Features / Aimlock Config | Allows defining specific players to ignore or target first. |
| **Enable FOV / Show FOV** | Features / FOV | Toggles the visual indicator for the aim FOV. |
| **FOV Color / FOV Type / FOV Size / FOV Stroke Thickness** | Features / FOV | Configuration options for the visual appearance and size of the FOV indicator. |
| **Rainbow FOV / Rainbow FOV Speed** | Features / FOV | Toggles and controls the speed of color cycling for the FOV indicator. |
| **FPS Cap / Apply FPS Cap** | Features / FPS | Limits or sets the game's frames per second (range shown: 60/720). |
| **Copy Server JobID / Copy Server Join Link** | Features / Server | Copies unique identifiers for the current server. |
| **Target Server JobID / Join Server / Rejoin Server** | Features / Server | Allows inputting an ID to join a specific server or instantly rejoining the current one. |
| **Rejoin Keybind** | Features / Server | Sets a keyboard shortcut for the Rejoin Server function. |
| **Quit Game** | Features / Server | Instantly closes the game application. |
| **Teleport / Select Player / Teleport To Player** | Features / Teleport | Enables teleportation to a selected player. |
| **Teleport Type / Noclip During Tween** | Features / Teleport | Configures the style of teleportation (e.g., Instant) and noclip during the move. |
| **Spectate / Spectate Player / Player To Spectate / Type** | Features / Spectate | Enables viewing other players, selecting the target, and setting the camera view (e.g., Third Person). |
| **Auto Chat / Auto Chat Delay / Auto Chat Message** | Features / Auto-Chat | Toggles and configures automatic sending of chat messages. |
| **Auto Chat Type / Times / Seconds** | Features / Auto-Chat | Controls how often the message is sent (e.g., Infinite, with a delay of 10 seconds). |
