## **Features Overview**

| Feature | Category / Tab | Description |
|:---|:---|:---|
| **Walkspeed** | Main / Character | Adjust player movement speed (shown range: 16–500). |
| **JumpPower** | Main / Character | Adjust player jump height (shown range: 50–1000). |
| **Max Zoom** | Main / Character | Set max camera zoom distance (shown range: 400–1000). |
| **Gravity** | Main / Character | Change world gravity (value shown: 196.199...). |
| **Infinite Jump** | Main / Character | Jump continuously without landing. |
| **Noclip** | Main / Character | Walk through solid objects. |
| **Force Third Person** | Main / Character | Lock camera in third-person view. |
| **X-Ray / Transparency (%)** | Main / Character | Makes objects transparent (shown: 80%–100%). |
| **Fly / Fly Speed** | Main / Fly | Allow free movement in air and adjust fly speed. |
| **Reset Character** | Main / Misc | Resets the player’s avatar and physics. |
| **Reset WalkSpeed/JumpPower/MaxZoom/Gravity** | Main / Misc | Reset individual settings to default. |
| **Field of View (FOV)** | Visuals / Game | Adjust camera FOV (shown range: 70–120). |
| **Full Bright / No Fog** | Visuals / Game | Enhance lighting for visibility. |
| **ESP** | Visuals / ESP | View entities through walls. |
| **Outline** | Visuals / ESP | Add outlines to players/entities. |
| **Tracers** | Visuals / ESP | Draw lines to targets. |
| **Outline Fill / Transparency (%)** | Visuals / UI | Control outline and fill transparency. |
| **Show Distance / Name** | Visuals / UI | Toggle player distance/name display. |
| **Tracers Position** | Visuals / UI | Select tracer anchor (e.g., Bottom). |
| **Rainbow ESP/Outline/Tracers** | Visuals / ESP Config | Color-cycling effects. |
| **Rainbow Speed** | Visuals / ESP Config | Adjust cycle speed. |
| **ESP Size / Font** | Visuals / ESP Config | Adjust ESP text appearance. |
| **Aimlock** | Features / Aimlock | Enable automatic aiming. |
| **Aimlock Type** | Features / Aimlock | Choose target mode (e.g., Nearest Character). |
| **Wall Check / Team Check** | Features / Aimlock | Filter invalid targets. |
| **Aimlock a Specific Player** | Features / Aimlock | Manually select target. |
| **Aimlock Max Dist / Mouse Dist / FOV Dist** | Features / Aimlock Config | Configure targeting ranges. |
| **Smooth Aimlock / Smoothness** | Features / Aimlock Config | Adjust aim smoothing. |
| **Aimlock Offset (X/Y)** | Features / Aimlock Config | Set aim offset (e.g., head/chest). |
| **Whitelist / Prioritize Players** | Features / Aimlock Config | Ignore or prioritize specific players. |
| **Enable FOV / Show FOV** | Features / FOV | Toggle aim FOV visibility. |
| **FOV Color / Type / Size / Stroke** | Features / FOV | Customize FOV visuals. |
| **Rainbow FOV / Speed** | Features / FOV | Color-cycling for FOV. |
| **FPS Cap / Apply** | Features / FPS | Limit FPS (range: 60–720). |
| **Copy Server JobID / Join Link** | Features / Server | Copy server info. |
| **Join by JobID / Rejoin / Join Server** | Features / Server | Server hop, rejoin, or target a server. |
| **Rejoin Keybind** | Features / Server | Assign a hotkey for rejoining. |
| **Quit Game** | Features / Server | Instantly exit the game. |
| **Teleport / Select Player** | Features / Teleport | Teleport to players. |
| **Teleport Type / Noclip During Tween** | Features / Teleport | Configure teleport methods. |
| **Spectate / Select Player / Type** | Features / Spectate | Spectate players with different camera modes. |
| **Auto Chat / Delay / Message** | Features / Auto-Chat | Automatically send messages. |
| **Auto Chat Type / Times / Seconds** | Features / Auto-Chat | Configure timed/infinite messaging. |
